# Ejercicio 5.1: Análisis de Datos con Excel y Python

> **Estado:** En construcción

## Descripción

Este ejercicio combina análisis de datos usando **Excel** y **Python** con la librería `openpyxl` y `pandas`.

Trabajarás con el archivo: `Ejercicio-de-Excel-resuelto-nivel-medio.xlsx`

---

## Objetivos

- Leer y procesar archivos Excel con Python
- Realizar análisis exploratorio de datos (EDA)
- Generar estadísticas y gráficos
- Comparar análisis manual en Excel vs automatizado con Python

## Requisitos

- Python 3.11+ con pandas, openpyxl, matplotlib
- Excel o LibreOffice Calc (para comparación manual)

**Instalación de librerías:**
