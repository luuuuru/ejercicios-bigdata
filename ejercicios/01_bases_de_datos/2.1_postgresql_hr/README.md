# Ejercicio 2.1: PostgreSQL con Base de Datos HR

> **Estado:** En construcción

---

## Descripción

Este ejercicio trabaja con la **base de datos HR de Oracle** adaptada a PostgreSQL.

**Base de datos HR:**
- Base de datos clásica de Oracle para gestión de recursos humanos
- Contiene tablas de empleados, departamentos, ubicaciones, salarios
- Perfecta para aprender consultas complejas y JOINs

---

## Objetivos

- Instalar y configurar PostgreSQL
- Cargar la base de datos HR en PostgreSQL
- Realizar consultas SQL avanzadas
- Comparar sintaxis SQL: Oracle vs PostgreSQL

---

## Contenido del Ejercicio

### Scripts Disponibles

- `scripts/01_crear_schema.sql` - Creación de tablas
- `scripts/02_cargar_datos.sql` - Inserción de datos
- `scripts/03_consultas_ejercicios.sql` - Ejercicios propuestos

### Datos

Los scripts SQL de la BD HR están en: `datos/oracle_hr/`

---

## Requisitos

- PostgreSQL 14+ instalado
- Cliente SQL (pgAdmin, DBeaver, o psql)
- Python 3.11+ con psycopg2 (opcional para scripts)
