# Ejercicio 3.2: Oracle con Base de Datos Jardinería

> **Estado:** En construcción

---

## Descripción

Este ejercicio adapta la **base de datos Jardinería** a Oracle Database.

Compararás la implementación en Oracle vs PostgreSQL (Ejercicio 2.2).

---

## Objetivos

- Adaptar esquemas de PostgreSQL a Oracle
- Utilizar características específicas de Oracle (sequences, triggers)
- Optimizar consultas en Oracle
- Comparar rendimiento con PostgreSQL

---

## Contenido del Ejercicio

### Scripts

- `scripts/schema_oracle.sql` - Esquema adaptado a Oracle
- `scripts/data_oracle.sql` - Datos de ejemplo
- `scripts/consultas.sql` - Ejercicios propuestos

---

## Requisitos

- Oracle Database 21c XE instalado
- SQL Developer o DBeaver
- Haber completado Ejercicio 2.2 (PostgreSQL Jardinería)
