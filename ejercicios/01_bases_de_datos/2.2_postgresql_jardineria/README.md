# Ejercicio 2.2: PostgreSQL con Base de Datos Jardinería

> **Estado:** En construcción

---

## Descripción

Este ejercicio trabaja con una **base de datos de una empresa de jardinería** que gestiona clientes, pedidos, productos y pagos.

**Base de datos Jardinería:**
- Sistema completo de gestión de ventas
- Tablas: clientes, pedidos, productos, oficinas, empleados
- Casos de negocio realistas para análisis

---

## Objetivos

- Practicar queries complejas en PostgreSQL
- Analizar datos de ventas y clientes
- Generar reportes de negocio con SQL
- Optimizar consultas con índices

---

## Contenido del Ejercicio

### Scripts Disponibles

- `scripts/schema.sql` - Estructura de la BD
- `scripts/data.sql` - Datos de ejemplo
- `scripts/consultas.sql` - Ejercicios propuestos

### Datos

Los scripts SQL están en: `datos/jardineria/`

---

- PostgreSQL 14+ instalado
- Cliente SQL (pgAdmin, DBeaver, o psql)
