# Ejercicio 3.1: Oracle con Base de Datos HR

> **Estado:** En construcción

---

## Descripción

Este ejercicio trabaja con la **base de datos HR original de Oracle**.

**Base de datos HR:**
- Base de datos de ejemplo oficial de Oracle
- Esquema completo de gestión de recursos humanos
- Ideal para aprender PL/SQL y características avanzadas de Oracle

---

## Objetivos

- Instalar y configurar Oracle Database (Express Edition)
- Trabajar con la BD HR en su entorno nativo
- Aprender sintaxis específica de Oracle
- Comparar con PostgreSQL (Ejercicio 2.1)

---

## Contenido del Ejercicio

### Scripts

- `scripts/hr_schema.sql` - Esquema original de Oracle
- `scripts/hr_data.sql` - Datos de ejemplo
- `scripts/consultas_oracle.sql` - Ejercicios con sintaxis Oracle

### Datos

Scripts en: `datos/oracle_hr/`
