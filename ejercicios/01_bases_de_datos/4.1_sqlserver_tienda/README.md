# Ejercicio 4.1: SQL Server con Tienda Informática

> **Estado:** En construcción

---

## Descripción

Este ejercicio migra la **base de datos de tienda informática** a Microsoft SQL Server.

Implementarás los mismos modelos del Ejercicio 1.1 pero en SQL Server.

---

## Objetivos

- Instalar y configurar SQL Server (Developer Edition)
- Adaptar esquemas SQLite/PostgreSQL a SQL Server
- Aprender sintaxis T-SQL
- Comparar con otros motores (SQLite, PostgreSQL, Oracle)

---

## Contenido del Ejercicio

### Scripts

- `scripts/schema_sqlserver.sql` - Estructura adaptada a T-SQL
- `scripts/data_sqlserver.sql` - Carga de datos
- `scripts/consultas_tsql.sql` - Ejercicios con T-SQL

---

## Requisitos

- SQL Server 2022 Developer Edition (gratuita)
- SQL Server Management Studio (SSMS)
- Windows o Docker para SQL Server
