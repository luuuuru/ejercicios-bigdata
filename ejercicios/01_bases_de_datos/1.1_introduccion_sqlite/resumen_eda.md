## 7. Conclusiones

### Entidades Identificadas para Modelo B (Normalizado)

1. **CATEGORIAS** - Una tabla con las categorías de productos
2. **FABRICANTES** - Una tabla con fabricantes únicos
3. **PRODUCTOS** - Tabla central con todos los productos
4. **COLORES** - Tabla con colores disponibles
5. **PRODUCTOS_COLORES** - Relación M:N entre productos y colores

### Relaciones Identificadas

- `CATEGORIAS` 1:N `PRODUCTOS` (una categoría tiene muchos productos)
- `FABRICANTES` 1:N `PRODUCTOS` (un fabricante fabrica muchos productos)
- `PRODUCTOS` M:N `COLORES` (un producto puede tener varios colores)

### Problema del Modelo A (Desnormalizado)

- 25 tablas independientes sin relaciones
- Fabricantes duplicados en cada tabla
- Difícil realizar consultas entre categorías
- Alta redundancia de información
