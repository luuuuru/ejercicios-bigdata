# Guia de Inicio

Bienvenido a la guia de inicio del curso de Big Data con Python. Esta seccion te guiara a traves de todo lo necesario para comenzar a trabajar con los ejercicios.

## Que encontraras aqui?

### [Instalacion de Herramientas](instalacion.md)

Aprende a instalar todas las herramientas necesarias para el curso:

- Python 3.11+
- Git
- PyCharm (IDE recomendado)
- Dependencias del proyecto

### [Tu Primer Ejercicio](primer-ejercicio.md)

Una guia paso a paso para completar tu primer ejercicio:

- Como navegar el repositorio
- Como ejecutar los ejercicios
- Como entregar tu trabajo
- Mejores practicas

### [Roadmap del Curso](roadmap.md)

Vision general de todos los ejercicios y tecnologias:

- Niveles de aprendizaje
- Tiempo estimado por ejercicio
- Tecnologias que dominaras
- Plan de estudio recomendado

## Por donde empezar?

!!! tip "Primera vez con Git y Python?"
    Empieza con la [Instalacion de Herramientas](instalacion.md) donde te explicamos paso a paso como instalar todo lo necesario.

!!! info "Ya tienes todo instalado?"
    Ve directo a [Tu Primer Ejercicio](primer-ejercicio.md) para comenzar a trabajar.

!!! warning "Desarrollador experimentado?"
    Revisa el [Roadmap del Curso](roadmap.md) para ver todos los ejercicios y elegir por donde empezar.
