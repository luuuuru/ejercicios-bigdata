# Carpeta de Entregas

Esta carpeta contiene las entregas de los alumnos organizadas por ejercicio.

## Estructura

```
entregas/
├── 1.1_sqlite/               # Entregas del ejercicio 1.1
│   ├── apellido_nombre/      # Carpeta de cada alumno
│   │   ├── ANALISIS_DATOS.md
│   │   ├── resumen_eda.md
│   │   └── REFLEXION.md
│   └── ...
└── [CODIGO_EJERCICIO]/       # Otros ejercicios
    └── apellido_nombre/
        └── ...
```

## Instrucciones para Alumnos

Cada ejercicio tiene sus propias instrucciones de entrega. Consulta el archivo `INSTRUCCIONES_ENTREGA.md` dentro de la carpeta del ejercicio correspondiente.

**Importante:**
- Usa tu apellido y nombre en minúsculas y separados por guión bajo
- Ejemplo: `garcia_maria`, `lopez_juan`, `martinez_ana`
- NO subas archivos de datos (.csv, .db, .sqlite)
- Solo sube los archivos solicitados en las instrucciones del ejercicio
